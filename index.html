UAV Sensor Spoofing Detection Project

Virginia Tech Senior Design / Research Project

Overview

This project focuses on the development of a virtual-to-physical UAV platform for generating, injecting, and detecting spoofed LiDAR and sensor data using Unreal Engine 5, Simulink middleware, and an embedded neural network running on NVIDIA Jetson hardware.

System Architecture

  • Unreal Engine 5 — Virtual LiDAR & spoof generation
  • Simulink Bridge — Timing, validation, and data injection
  • Hardware Abstraction Layer — Sensor bus emulation
  • Edge AI (Jetson) — Real-time spoof detection
