# About the Project

## Mission
The mission is to improve awareness of sensor spoofing threats while developing a practical platform for neural network-based detection in UAV systems. This project leverages **Unreal Engine 5** to create a high-fidelity simulation environment capable of generating realistic sensor spoofing scenarios and supporting AI-based detection development.

The goal is not just simulation, but enabling repeatable, controlled experiments that are difficult to achieve in real-world testing, especially for adversarial conditions.

This project builds upon [Project AirSim](https://github.com/iamaisim), maintained by **IAMAI Simulations**, a team of former engineers from Microsoft's original AirSim project. The base framework has been extended with custom scripts, libraries, and Unreal Engine 5 modifications to support spoofing workflows and detection pipelines.

All credit for the original AirSim framework goes to IAMAI Simulations and Microsoft. Our contributions focus on spoofing injection methods, dataset generation, and detection system integration within our forked implementation.

## Scope
- Virtual sensor generation across LiDAR and camera systems  
- Controlled spoof injection for adversarial scenario testing  
- Dataset generation for supervised and unsupervised learning  
- Embedded AI deployment for real-time classification  
- Emulated drone flight and testing environments  
- Modular system architecture for extensibility and future research  

## Impact
This platform enables rapid prototyping and evaluation of spoof detection techniques without requiring expensive or high-risk field testing. By combining simulation, adversarial modeling, and embedded AI, the project provides a foundation for more resilient UAV perception systems.

The long-term goal is to bridge simulation and real-world deployment by validating detection strategies in a controlled environment before transitioning to hardware-in-the-loop and live system testing.
